# component-editor

> 本项目是一个大屏编辑器，目的是让业务能够快速开发报表，可以使用常见的组件以及部分图表，让数据可视化。

## 技术栈

+ Vue
+ Vuex
+ ElementUi
+ Echarts
+ Vue-draageble-resize
+ vue-color

## 预览

![](https://github.com/cadben/images_store/blob/master/img/%E6%A1%88%E4%BE%8B.png?raw=true)

## 支持以下几种图表

+ 柱状图
+ 折现图
+ 面积图
+ 漏斗图
+ 饼图

## 数据支持

+ 静态数据
+ API接口
